## Overview

The ARDA Seeds website serves as the digital presence for ARDA Seeds (PVT) Ltd, providing Zimbabwean farmers and agricultural stakeholders with:

- A comprehensive product catalogue of seed varieties
- Online quote requests and seed ordering
- Agronomic support and resources
- Company news, careers, and tender listings
- Grower registration and outgrower program information

## Getting Started

### Prerequisites

Ensure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

```bash
# Clone the repository
git clone https://github.com/craigtanatswa/arda-seeds-website.git

# Navigate to the project directory
cd arda-seeds-website

# Install dependencies
npm install
```

### Running Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.

### Building for Production

```bash
npm run build
npm run start
```

## Deployment

This project is deployed on **[Vercel](https://vercel.com)**. Any push to the main branch will trigger an automatic deployment.

To deploy your own instance:

1. Push the project to a GitHub repository
2. Import the repository on [vercel.com](https://vercel.com)
3. Vercel will auto-detect the framework and deploy
