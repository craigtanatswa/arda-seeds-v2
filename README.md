Overview
The ARDA Seeds website serves as the digital presence for ARDA Seeds (PVT) Ltd, providing Zimbabwean farmers and agricultural stakeholders with:

A comprehensive product catalogue of seed varieties
Online quote requests and seed ordering
Agronomic support and resources
Company news, careers, and tender listings
Grower registration and outgrower program information

Getting Started
Prerequisites
Ensure you have the following installed:

Node.js (v18 or higher recommended)
npm or yarn

Installation
bash# Clone the repository
git clone https://github.com/your-username/arda-seeds-website.git

# Navigate to the project directory
cd arda-seeds-website

# Install dependencies
npm install
Running Locally
bashnpm run dev
Open http://localhost:3000 in your browser to view the site.
Building for Production
bashnpm run build
npm run start

Deployment
This project is deployed on Vercel. Any push to the main branch will trigger an automatic deployment.
To deploy your own instance:

Push the project to a GitHub repository
Import the repository on vercel.com
Vercel will auto-detect the framework and deploy
